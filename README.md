# English Course — Landing (Hero Screen)

Односторінковий hero-екран для курсу англійської мови з
кінематографічною 3D-анімацією: камера летить крізь хмару літер,
які збираються у книгу, паперовий літак і чашку кави, перетворюються
на слова, слова — на речення, речення — на діалог, діалог — на
заголовок з кнопками.

Технології: Three.js (WebGL) + GSAP, без збірки — чистий HTML/CSS/JS.

## Як запустити локально

Просто відкрий `index.html` у браузері. Інтернет потрібен лише для
завантаження бібліотек Three.js та GSAP з CDN.

## Як опублікувати на GitHub Pages

1. Запуш цей репозиторій на GitHub.
2. В репозиторії відкрий Settings -> Pages.
3. У розділі "Build and deployment" вибери Source: "Deploy from a branch".
4. Branch: `main`, папка `/ (root)` -> Save.
5. Через 1-2 хвилини сайт буде доступний за адресою
   `https://<твій-логін>.github.io/<назва-репозиторію>/`.

## Що редагувати

Усі тексти, посилання та кольори — в об'єкті `CONFIG` на початку
блоку `<script>` в `index.html`. Детальний опис кожного поля — у
коментарях `/* ЗМІНИТИ */` поруч із кожним параметром.
